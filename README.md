> # No Crystal Hitbox

**Clean up your Crystal PvP experience** by selectively hiding End Crystal hitboxes while keeping player hitboxes visible! 

> ## Features
  - Hides only **end crystals** hitboxes, keeps everything else visible
  - Toggle keybinds (default H)
